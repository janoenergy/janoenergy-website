## 🎨 配色方案

### 主色调 (品牌色)
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#2AB8A8` / `#42C4B8` | 青绿色，用于背景、强调 |
| 深色 | `#1A8B7D` / `#298E80` | 深青绿，用于渐变底部 |
| 浅色 | `#4BC7BB` | 浅青绿，用于渐变顶部 |

### 辅助色
| 用途 | 色值 | 说明 |
|------|------|------|
| 定位/红色 | `#E85A5A` / `#F07070` | 珊瑚红，用于定位板块 |
| 使命/绿色 | `#2AB8A8` | 主色，用于使命板块 |
| 愿景/黄色 | `#F5B942` / `#FFC857` | 金黄色，用于愿景板块 |
| 数据/橙色 | `#F5A623` | 橙色，用于数据强调 |

### 中性色
| 用途 | 色值 | 说明 |
|------|------|------|
| 背景白 | `#F9FAFE` / `#FFFFFF` | 纯白背景 |
| 文字黑 | `#1A1A1A` / `#333333` | 主文字色 |
| 文字灰 | `#666666` / `#888888` | 次要文字 |
| 线条灰 | `#E5E5E5` / `#CCCCCC` | 分割线、边框 |

## 📝 字体方案

### 中文字体
- **标题**: 思源黑体 / Noto Sans SC (粗体)
- **正文**: 思源黑体 / Noto Sans SC (常规)

### 英文字体
- **标题**: Montserrat / Arial (粗体)
- **正文**: Open Sans / Arial (常规)

### 字号层级
| 层级 | 用途 | 参考字号 |
|------|------|----------|
| 超大 | 章节号 (01/02/03) | 200pt+ |
| 大标题 | 页面主标题 | 36-48pt |
| 中标题 | 板块标题 | 24-32pt |
| 小标题 | 内容标题 | 18-24pt |
| 正文 | 说明文字 | 14-16pt |
| 注释 | 英文副标题 | 12-14pt |

## 🖼️ 版式布局

### 封面页
- 全屏渐变背景 (青绿 `#42C4B8` → `#298E80`)
- 左上角 Logo (Jano Power)
- 中央大标题 + 年份标签
- 右下角装饰图形 (风力发电机)
- 底部副标题 (中英文)

### 目录页
- 左侧 70% 白色背景 + 目录内容
- 右侧 30% 品牌色背景 + Logo
- 章节编号 (01/02/03/04) + 标题 + 英文

### 章节页
- 全白背景
- 超大章节号 (渐变效果: 青绿 → 透明)
- 章节标题居中
- 右上角 Logo

### 内容页
- 白色背景为主
- 顶部标题栏 (中文标题 + 英文副标题 + Logo)
- 内容区域灵活布局
- 常用结构:
  - 三栏卡片式 (定位/使命/愿景)
  - 时间轴 (发展历程)
  - 数据图表

## 🎯 设计元素

### Logo 使用
- 位置: 右上角 (内容页) / 左上角 (封面)
- 版本: 白色版 (深色背景) / 彩色版 (白色背景)
- 尺寸: 保持比例，高度约 40-60pt

### 图标风格
- 线性图标 (outline)
- 圆角处理
- 品牌色填充

### 图形元素
- 渐变填充 (青绿色系)
- 圆角矩形 (8-16px 圆角)
- 轻微阴影 (透明度 10-20%)

### 时间轴样式
- 曲线连接线 (灰色 `#CCCCCC`)
- 节点圆点 (品牌色 `#2AB8A8`)
- 年份标签 (品牌色 + 粗体)
- 里程碑标题 (黑色 + 关键数据橙色)

## 📋 应用建议

### 新建 PPT 时
1. 设置页面 16:9 比例
2. 定义主题色 (主色 `#2AB8A8`)
3. 设置默认字体 (思源黑体)
4. 创建母版页 (封面/目录/章节/内容)

### 配色应用
- 背景: 白色 或 品牌色渐变
- 标题: 黑色 或 白色 (视背景)
- 强调: 品牌青绿 或 辅助色
- 数据: 橙色强调关键数字

### 排版原则
- 保持留白 (页边距 40-60pt)
- 对齐统一 (左对齐或居中对齐)
- 层次分明 (字号对比 1.5-2x)
- 图文平衡 (图片占比 30-50%)
