# 代码质量改进项目

## 项目名称
**JanoEnergy 网站代码质量提升项目**

## 项目目标
将代码质量评分从 **5.95/10** 提升至 **8.5/10**

## 项目周期
**4 周**（2026-03-19 至 2026-04-16）

---

## 一、项目背景

### 当前问题
1. 代码质量评分 5.95/10，低于及格线
2. 79 处 `any` 类型，类型安全缺失
3. 5 个组件超过 500 行，维护困难
4. 81 处重复图片 URL，管理混乱
5. 类型定义分散，重复定义 6 次

### 改进收益
- 提升代码可维护性
- 减少 Bug 率
- 提高开发效率
- 增强系统稳定性

---

## 二、项目范围

### 包含内容
- 类型系统重构
- 组件拆分优化
- 代码复用提升
- 性能优化
- 安全加固

### 不包含内容
- 功能新增
- UI  redesign
- 后端 API 改造
- 数据库迁移
