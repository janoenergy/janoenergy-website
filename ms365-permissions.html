🔐 Microsoft Graph 权限配置

为 OpenClaw 添加所需的 Microsoft 365 API 权限

1 打开 Azure Portal

点击下方按钮直接进入应用注册页面的 API 权限配置：

打开 Azure Portal → API 权限
2 添加应用权限

点击 "添加权限" → "Microsoft Graph" → "应用程序权限"，然后添加以下权限：

需要添加的权限：

  • User.Read.All - 读取所有用户
  • Mail.ReadWrite - 读写邮件
  • Mail.Send - 发送邮件
  • Calendars.ReadWrite - 读写日历
  • Sites.Read.All - 读取 SharePoint 站点
3 授予管理员同意

添加权限后，点击 "授予管理员同意" 按钮（需要管理员权限）

提示： 只有租户管理员才能授予应用程序权限的同意。
4 完成

权限配置完成后，告诉我一声，我来测试新权限是否生效！
